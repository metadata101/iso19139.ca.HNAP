Visualisation des données

Tous les employés peuvent utiliser le catalogue de données pour rechercher et découvrir les données environnementales et scientifiques. Pour commencer, il suffit de naviguer jusqu’à la page d’accueil du Catalogue de données et de sélectionner un type de recherche.

Aperçu

Les données géospatiales du catalogue de données peuvent être visualisées à l’aide de la fonction interactive de visualisation de cartes Web.

Processus

Découvrir > Combiner les données/Sélectionner les données > Visualiser

Formation connexe

Découverte de données, accès aux données

Avant de commencer

Il n’y a aucune condition à l’utilisation de cette fonction. Quiconque accède au catalogue de données d’ECCC peut visualiser les données géospatiales avec le visualiseur de cartes.

Découvrez

  1. Pour commencer à visualiser des données géospatiales, il suffit de localiser les données à l’aide de la fonction de recherche du catalogue de données (pour plus d’information, voir Découvrir des données).

  2. Si un ensemble de données peut être visualisé à l’aide du visualiseur de cartes, vous verrez deux boutons supplémentaires : un globe terrestre ou un globe terrestre avec le menu déroulant et l’épinglette avec le menu déroulant.

    Ceux-ci vous permettent de :

    • vue sur carte, qui ouvrira l’ensemble de données sélectionné dans le visualiseur de cartes, ou

    • choisir les calques à afficher sur la carte, ce qui ouvrira les calques sélectionnés dans le visualiseur de la carte, ou

    • ajouter l’ensemble de données à la carte, où vous pouvez combiner plusieurs ensembles de données pour les afficher dans une seule carte.

Afficher les données

  1. Pour afficher un seul ensemble de données dans le visualiseur de cartes, cliquez simplement sur le bouton « Afficher sur la carte » pour cet ensemble de données. Le visualiseur de cartes s’ouvre et affiche les données.

    [screenshot of search result, arrow points to view on map button]

Combiner les données

  1. En ajoutant des ensembles de données à l’aperçu de la carte, vous pouvez superposer plusieurs ensembles de données dans un seul visualiseur. Cela peut être utilisé pour générer de nouvelles informations et soutenir la prise de décision.

    [capture d’écran de l’exemple affiché dans le visualiseur de la carte]

    Ajoutez des ensembles de données à la liste de prévisualisation en cliquant sur le bouton « Ajouter à la liste de cartes » (voir l’icône de l’épinglette) sur chaque ensemble de données à partir de la page des résultats de recherche ou des détails des métadonnées.

    [capture d’écran du résultat de recherche et flèche pour ajouter au bouton de prévisualisation de la carte]

  2. La liste « Ajouter à la carte » avec les ensembles de données que vous avez sélectionnés s’affichera à gauche de la page des résultats de recherche. À tout moment, vous pouvez :

    [screenshot of map preview list]

    • supprimer les données que vous avez ajoutées à l’aperçu en cliquant sur le X à côté du titre

    • combiner et visualiser les jeux de données en cliquant sur “Ajouter à la carte”. Cliquez sur l’onglet “Carte” dans le menu d’administration pour afficher les résultats.
